# FFUF

FFUF — это мощный и быстрый инструмент для фаззинга веб-приложений. Он используется для тестирования безопасности и поиска уязвимостей в веб-приложениях. FFUF может сканировать веб-сайты, API и другие веб-приложения, находя и выявляя уязвимые места. Он поддерживает множество параметров для настройки сканирования, таких как URL-адрес, метод запроса, заголовки, параметры запроса и многое другое. Изучение документации FFUF позволит получить максимальную эффективность от этого инструмента.
## Установка

Чтобы установить FFUF на Linux, нужно выполнить следующие шаги:

    Откройте терминал и выполните команду `sudo apt update`, чтобы обновить список пакетов.
    Установите Go, если он еще не установлен, с помощью команды `sudo apt install golang-go`.
    Выполните команду `go get -u github.com/ffuf/ffuf`, чтобы установить FFUF.
    После установки FFUF вы можете проверить правильность установки с помощью команды `ffuf -h`.

Теперь вы можете использовать FFUF для тестирования безопасности и поиска уязвимостей в веб-приложениях.
Примеры

После установки данной программы, чтобы узнать ее параметры, все, что нужно сделать,- это просто использовать опцию «-h» для вызова справки.

Примеры:

    Простое сканирование веб-сайта с использованием 40 потоков: `ffuf -u <https://example.com/FUZZ>`
    Сканирование веб-сайта, используя список URL-адресов из файла: `ffuf -w /path/to/wordlist.txt -u <https://example.com/FUZZ>`
    Сканирование веб-сайта, используя метод POST: `ffuf -X POST -d "username=admin\\\\&password=FUZZ" -u <https://example.com/login>`
    Сканирование веб-сайта, используя кастомные заголовки: `ffuf -H "Authorization: Bearer FUZZ" -u <https://example.com/api>`

Используя список параметров FFUF, вы можете настроить запросы на свой вкус. Например, вы можете использовать параметр -w для загрузки списка слов из файла и использовать их для фаззинга. Вы также можете использовать параметр -u для указания URL-адреса, который вы хотите сканировать, и параметр -t для установки количества одновременных потоков.

Также, если вам нужно сканировать веб-сайт по определенным критериям, вы можете использовать параметры, такие как -H для задания заголовков запроса, -X для указания метода HTTP-запроса, -d для отправки POST данных, а -s для настройки запроса в формате ключ=значение.

Пример использования FFUF:

Сканирование веб-сайта, используя параметры
`ffuf -u <https://example.com/FUZZ> -w /path/to/wordlist.txt -t 50 -H "Authorization: Bearer TOKEN" -X POST -d "username=admin\\&password=FUZZ"`

В этом примере мы используем -u для задания URL-адреса, который мы хотим сканировать, -w для загрузки списка слов из файла, -t для установки количества одновременных потоков, -H для задания кастомных заголовков запроса, -X для указания метода HTTP-запроса, и -d для отправки POST данных.
